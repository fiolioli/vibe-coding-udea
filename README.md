# Semillero Vibe Coding Universidad de Antioquia

Semillero estudiantil de la Universidad de Antioquia enfocado en desarrollo asistido por IA: vibe coding, meta prompting, ingeniería de prompts y modelos locales.

## Encuentros
- **Ubicación:** Bloque 18-210 (LIS), Sala 4.
- **Horario:** Miércoles, 12:30 – 13:30.

## Metodología
- Trabajo autónomo coordinado de manera presencial y digital.
- Apoyo entre pares con perspectiva crítica.
- Desarrollo de proyectos personales y colaborativos.
- Colaboración mediante prácticas de código abierto.

## Objetivos
- Potenciar perfiles profesionales y académicos mediante IA.
- Desarrollar herramientas y flujos de datos que vinculen IA con áreas de estudio específicas.
- Colaborar y asistir técnicamente a otros estudiantes en el uso táctico de IA.

## Colaboración
- Únete como colaborador del repositorio.
- Contribuye mediante Pull Requests.
- Utiliza "Conventional Commits" para los mensajes de commit.
- Comparte tus hallazgos y vincula tus proyectos.
