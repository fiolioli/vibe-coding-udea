# Editores y asistentes

* **Google Antigravity**
    * **Descripción:** IDE de Google con ejecución paralela de hasta 5 agentes.
    * **Característica clave:** Sistema de "Artifacts" e integración profunda con Gemini 3.1 Pro.
    * **Enlace oficial:** [https://antigravity.google](https://antigravity.google)

* **Cursor**
    * **Descripción:** Editor nativo de IA basado en VS Code con edición de múltiples archivos.
    * **Característica clave:** Indexación de código veloz y "Tab-to-complete++" con Claude 4/GPT-5.
    * **Enlace oficial:** [https://www.cursor.com/](https://www.cursor.com/)

* **Windsurf (de Codeium)**
    * **Descripción:** IDE para flujos agentiles de larga duración con contexto persistente.
    * **Característica clave:** Agente "Cascade" altamente autónomo.
    * **Enlace oficial:** [https://codeium.com/windsurf](https://codeium.com/windsurf)

* **VS Code con Copilot**
    * **Descripción:** Editor estándar extensible con integración nativa al ecosistema de GitHub.
    * **Característica clave:** Soporte masivo e integración con seguridad de GitHub.
    * **Enlaces oficiales:** [VS Code](https://code.visualstudio.com/) | [Copilot](https://github.com/features/copilot)

* **Zed**
    * **Descripción:** Editor en Rust optimizado para velocidad y rendimiento nativo.
    * **Característica clave:** Chat de IA integrado con soporte para múltiples backends.
    * **Enlace oficial:** [https://zed.dev/](https://zed.dev/)

* **Claude Code**
    * **Descripción:** Interfaz de terminal de Anthropic para razonamiento y migraciones complejas.
    * **Característica clave:** Ventana de contexto de 1 millón de tokens.
    * **Enlace oficial:** [https://claude.ai/code](https://claude.ai/code)

* **JetBrains AI Assistant**
    * **Descripción:** Asistente empresarial con análisis estático profundo.
    * **Característica clave:** Sugerencias "type-aware" para código orientado a objetos complejo.
    * **Enlace oficial:** [https://www.jetbrains.com/ai/](https://www.jetbrains.com/ai/)
